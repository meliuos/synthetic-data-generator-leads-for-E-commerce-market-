  <action>
In dashboard/app.py, add the following after any existing page/section setup:

1. **Add screenshot service client function:**
   ```python
   import requests
   import os
   from pathlib import Path
   
   SCREENSHOT_SERVICE_URL = os.getenv("SCREENSHOT_SERVICE_URL", "http://playwright-screenshot:8000")
   
   def capture_screenshot(url: str) -> dict:
       """Call screenshot service to capture URL at both viewports."""
       try:
           response = requests.post(
               f"{SCREENSHOT_SERVICE_URL}/capture",
               json={"url": url},
               timeout=30
           )
           response.raise_for_status()
           return response.json()
       except Exception as e:
           st.error(f"Failed to capture screenshot: {e}")
           return None
   
   def get_screenshot_path(url_hash: str, viewport: int) -> Path:
       """Return path to cached screenshot."""
       return Path("./screenshots") / url_hash / f"{viewport}.png"
   ```

2. **Add UI section in main app:**
   ```python
   st.subheader("📸 Page Screenshot")
   
   # URL selector
   page_urls = [
       "https://example.com",
       "https://example.com/products",
       "https://example.com/blog"
   ]
   selected_url = st.selectbox("Select page to view:", page_urls)
   
   # Refresh button
   col1, col2 = st.columns(2)
   with col1:
       refresh_button = st.button("🔄 Refresh Screenshot", key="refresh_btn")
   with col2:
       st.text(f"Last updated: {st.session_state.get('screenshot_timestamp', 'Never')}")
   
   # Refresh action
   if refresh_button:
       with st.spinner("Capturing screenshot..."):
           result = capture_screenshot(selected_url)
           if result:
               st.session_state.screenshot_timestamp = datetime.now().strftime("%H:%M:%S")
               st.success(f"Screenshot captured (cached: {result.get('cached')})")
               st.rerun()
   
   # Display screenshots
   st.write("---")
   
   try:
       # Get screenshot paths from service by hashing URL
       import hashlib
       url_hash = hashlib.sha256(selected_url.encode()).hexdigest()[:12]
       
       desktop_path = get_screenshot_path(url_hash, 1440)
       mobile_path = get_screenshot_path(url_hash, 390)
       
       # Tabs for viewport selection
       tab1, tab2 = st.tabs(["Desktop (1440px)", "Mobile (390px)"])
       
       with tab1:
           if desktop_path.exists():
               st.image(str(desktop_path), use_column_width=True)
           else:
               st.info("Click 'Refresh Screenshot' to capture desktop view")
       
       with tab2:
           if mobile_path.exists():
               st.image(str(mobile_path), use_column_width=True)
           else:
               st.info("Click 'Refresh Screenshot' to capture mobile view")
               
   except Exception as e:
       st.error(f"Failed to load screenshot: {e}")
   ```

3. **Initialize session state:**
   At the very top of app.py, add:
   ```python
   from datetime import datetime

       st.session_state.screenshot_timestamp = None
   ```

4. **Error handling:** Gracefully handle cases where service is unreachable — show message directing user to start service. Do NOT crash dashboard if screenshot service is down.
