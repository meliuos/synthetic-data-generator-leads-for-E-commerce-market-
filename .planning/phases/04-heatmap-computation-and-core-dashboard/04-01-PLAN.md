Output: Reusable query helpers that return heatmap-ready dataframes plus the supporting dashboard dependency update.

Create a small query module that encapsulates the Phase 4 aggregation contract.

Implement helpers that:
- Accept a page URL filter that supports exact matches and wildcard patterns such as `/product/*`.
- Accept an event type selector for click, scroll, and mousemove views.
- Aggregate `x_pct` and `y_pct` into 5% bins using ClickHouse `round(... / 5) * 5` semantics.
