4. **CREATE OR REPLACE MATERIALIZED VIEW analytics.events_mv TO analytics.click_events AS ÔÇª** ÔÇö the updated main MV. 6. **CREATE MATERIALIZED VIEW IF NOT EXISTS analytics.orders_mv TO analytics.orders AS ÔÇª** ÔÇö exactly as shown in RESEARCH.md ┬º5. Same WHERE filter as purchase_items_mv. Extracts total/revenue/tax/shipping/discount as `Nullable(Decimal(18,2))`, preserves raw products JSON string.

7. **CREATE OR REPLACE VIEW analytics.click_events_ga4 AS ÔÇª** ÔÇö GA4 alias VIEW (not materialized column) as shown in RESEARCH.md ┬º6. Aliases: `product_id AS item_id`, `category AS item_category`, `order_id AS transaction_id`, `search_query AS search_term`. Zero-storage read-time remapping.
