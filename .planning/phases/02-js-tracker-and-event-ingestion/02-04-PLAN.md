  </what-built>
  <how-to-verify>
    Follow these steps to verify the end-to-end pipeline:
    
    **Setup:**
    1. Ensure Phase 1 stack is running: `make up && make schema`
    2. Open browser to http://localhost:5000/test-spa-page.html (or where your test page is served)
    
    **Test Sequence:**
    1. **Consent Gating (Criterion 5):**
       - Page loads with cookie banner visible
       - Click on an element WITHOUT accepting consent
       - Open DevTools -> Console -> Check if event queue is empty
       - Expected: No events captured
       
    2. **Click with Coordinates (Criterion 1):**
       - Scroll down 500px on the page
       - Note your approximate position on the scrollbar
       - Click on a visible button or element
       - Accept consent when banner appears
       - Wait 5 seconds
       - Run in ClickHouse: `SELECT x_pct, y_pct, event_type FROM analytics.click_events WHERE event_type='click' ORDER BY event_time DESC LIMIT 1`
       - Expected: x_pct and y_pct values between 0-100, matching roughly where you clicked
       
    3. **Scroll Depth (Criterion 2):**
       - Scroll to approximately 75% down the page (use scrollbar as visual cue)
       - Wait 5 seconds
       - Run in ClickHouse: `SELECT scroll_pct, event_type FROM analytics.click_events WHERE event_type='scroll' ORDER BY event_time DESC LIMIT 1`
       - Expected: scroll_pct value around 70-80
       
    4. **Mouse Movement Throttling (Criterion 3):**
       - Move mouse over the page for 10 seconds quickly back and forth
       - Wait 5 seconds
       - Run in ClickHouse: `SELECT COUNT(*) as mousemove_count FROM analytics.click_events WHERE event_type='mousemove' AND event_time > NOW() - INTERVAL 15 second`
       - Expected: count should be around 100-150 (10-15 events per second if 10 seconds, throttled to <10/sec means ~100 max)
       
    5. **SPA Route Change (Criterion 4):**
       - Click navigation button to change route (e.g., /page2)
       - Wait 5 seconds
       - Run in ClickHouse: `SELECT event_type, url FROM analytics.click_events WHERE event_type='page_view' ORDER BY event_time DESC LIMIT 2`
       - Expected: Two recent page_view events with different URLs (/page1, /page2, etc.)
    
    **If All Tests Pass:**
    - Type: "approved" to proceed to planning Phase 3
    
    **If Any Test Fails:**
    - Describe which criterion failed and what you observed
    - Include the ClickHouse query result if available
  </how-to-verify>
  <resume-signal>Type "approved" if all tests pass. Describe failure details if any test does not pass.</resume-signal>
</task>
